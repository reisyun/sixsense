## How can I see the work?

### 1. SSWEB Project 복제

```bash
$ git clone https://github.com/J2KB/SixSense-SSWEB.git
```

### 2. Dependency를 설치합니다

```bash
$ cd SixSense-SSWEB/ssweb-client

# or npm install
$ yarn
```

### 3. 이제 로컬에서 확인하실 수 있습니다!

```bash
# or npm run start:storybook
$ yarn start:storybook
```
